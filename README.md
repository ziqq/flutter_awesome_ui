# flutter_awesome_ui

## Introduction
Package created to give a fully functional and customizable: Text Form Field, Select.


```dart
AwesomeInput(
 labelText: '',
 controller: controller,
 onChanged(String? value) {},
),

AwesomeSelect(
  labelText: '',
  initialValue: 1,
  items: [],
),
```
